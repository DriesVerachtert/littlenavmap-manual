# ![Little Navmap](../images/littlenavmap.svg "Little Navmap") Little Navmap用户手册 {#little-navmap-user-manual}

**Version 2.0**

_Little Navmap_ 是一个免费、开源的软件，包含飞行计划，导航工具，移动地图，
机场搜索和机场信息系统等功能，同时支持FSX，FSX steam版，P3d v2到v4和X- plane 11。

**请勿使用此软件用作真实世界的导航**
